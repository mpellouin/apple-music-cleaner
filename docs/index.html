apple-music-cleaner

Bulk-remove Apple Music favorites or wipe your library using web-player session tokens. No paid Apple Developer account.

  • GitHub repository
  • Documentation (README)
  • llms.txt — AI-friendly index

Quick start

git clone https://github.com/mpellouin/apple-music-cleaner
cd apple-music-cleaner && npm install
npm start -- --preset stale-favorites
npm run web
